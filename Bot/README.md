# 🤖 해정뉴스 자동 테스트 봇 v2.0

## 📦 실행 방법

### Windows
`run.bat` 파일을 더블클릭 (또는 CMD에서 실행)

### Mac / Linux
```bash
chmod +x run.sh && ./run.sh
```

### VSCode 터미널
```
cd Bot
.\run.bat    ← CMD 창에서 (PowerShell ❌)
```

---

## ⚙️ 로그인 계정 설정 (선택)

`Bot.js` 상단 `CONFIG` 섹션에 테스트용 계정을 입력하면
**로그인이 필요한 모든 기능도 자동으로 테스트**됩니다.

```js
TEST_EMAIL:    'bot@test.com',   // Firebase 이메일/비밀번호 계정
TEST_PASSWORD: 'botpass123',
```

> ⚠️ Google 소셜 로그인 계정은 사용 불가합니다.
> Firebase Console → Authentication → 이메일/비밀번호 로그인을 활성화한 뒤
> 테스트 전용 계정을 별도로 만들어 입력하세요.

계정 미설정 시 → 비로그인으로 가능한 항목만 테스트 (⚠️ WARN 처리)
