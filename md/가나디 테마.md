### ✨ 보너스 가나디 효과
특정 행동을 하면 화면 중앙 부근에 가나디가 팝업으로 나타났다가 사라집니다!

| 행동 | 확률 | 사용 이미지 | 설명 |
|------|------|------------|------|
| **좋아요** | 80% | ganadi4.png | 좋아요 버튼 클릭 시 |
| **댓글 작성** | 100% | ganadi5.png | 댓글 작성 완료 시 |

**애니메이션**: 
- 0초: 투명, 작게 시작, 회전 0도
- 0.3초: 불투명, 130% 확대, 회전 15도
- 0.75초: 불투명, 100% 크기, 회전 -10도
- 1.5초: 투명, 50% 축소, 위로 100px 이동, 회전 20도

## 🔧 개선된 기능

### ✅ 깔끔한 가나디
1. **고정 코너 장식**: 네 코너에 작고 은은하게 배치
2. **자동 복원**: 1초마다 가나디 상태를 체크하여 사라졌으면 자동으로 다시 생성
3. **페이지 전환 대응**: 페이지 이동 시에도 가나디가 유지됨
4. **테마 감지**: body 클래스 변경을 실시간으로 감지

### 🎯 사용 방법

```javascript
// 1. 수동으로 보너스 효과 표시
showGanadiBonusEffect('like');

// 2. 빈 화면 표시
showGanadiEmpty('articlesList', '🐶 듀... 기사가 없어요');

// 3. 로딩 표시
showGanadiLoading('articlesList');

// 4. 가나디 강제 업데이트
updateGanadiElements();

// 5. 현재 테마 확인
if (isGanadiTheme()) {
    console.log('가나디 테마입니다!');
}
```

### 이미지가 없을 때
- 각 이미지 로드 실패 시 자동으로 `ganadi1.png`로 폴백됩니다
- 또는 해당 요소가 숨겨집니다

### 테스트 방법
1. 가나디 테마 선택
2. 화면 네 코너에 작은 가나디들이 은은하게 보이는지 확인
3. 마우스를 코너 가나디에 올리면 살짝 커지고 진해지는지 확인
4. 기사 카드에 마우스 올리면 우상단에 작은 가나디가 살짝 나타나는지 확인
5. 좋아요나 댓글 작성하면 보너스 가나디가 팝업되는지 확인
6. 새로고침해도 가나디가 유지되는지 확인

### 문제 해결
- 가나디가 안 보인다면: `updateGanadiElements()` 콘솔에서 실행
- 보너스 효과가 안 나온다면: `showGanadiBonusEffect('like')` 실행해보기
- 테마가 바뀌었는지 확인: `isGanadiTheme()` 콘솔에서 실행
