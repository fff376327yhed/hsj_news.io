해정뉴스 개발 스토리

AI와 함께 만든 이 웹사이트, 어떻게 탄생했을까요?

바이브 코딩(Vibe Coding)이란?

가장 궁금해하셨을 제작 과정부터 설명하겠습니다. 혹시 '바이브 코딩'이라고 들어보셨나요?

💡 바이브 코딩이란?
개발자가 직접 한 땀 한 땀 코드를 짜는 것이 아니라, AI가 주 개발자가 되어 코드를 작성하고 사람은 그것을 지휘하는 방식입니다.

이 방식의 장점은 코딩을 깊게 몰라도 기초적인 지식만 있다면 누구나 쉽게 원하는 것을 제작할 수 있다는 점입니다. 물론 단점도 존재합니다. AI가 짜주는 코드에 오류가 있거나, 내가 의도한 대로 결과물이 나오지 않을 수도 있습니다.


>

코딩의 기초 & 도구

우리가 코드를 짜기 위해서는 기본적으로 코딩 도구(Tool)가 필요합니다. 모바일 환경은 제작이 힘드니 PC 환경 기준으로 설명합니다.

🛠️ 필수 도구

  • 1. Visual Studio Code (VSC)
    가장 대중적인 코드 편집기입니다. 다운로드 후 설정에서 한국어 패치를 하시면 편합니다.
  • 2. Live Server (확장 프로그램)
    VSC의 '확장(Extensions)' 탭에서 필수적으로 설치해야 합니다. 내가 짠 코드를 실시간으로 웹에서 볼 수 있게 해줍니다.
HTML 웹사이트의 뼈대
(글, 그림 등)
CSS 디자인 및 꾸미기
(색상, 배치)
JavaScript 기능 구현
(동작, 클릭 이벤트)

보통 index.html, style.css, script.js로 파일을 만들어 시작합니다.

AI와 협업하기 (프롬프트)

기초는 알았지만, 백지에서 코드를 짜기는 막막합니다. 그래서 우리는 AI의 도움을 받습니다.

ChatGPT, Claude 등 다양한 모델이 있지만, 2025.11.21 기준 가장 추천하는 것은 Gemini입니다. 다른 AI들은 코드가 길어지면 중간에 잘리거나 오류가 나기 일쑤인데, Gemini는 긴 코드도 척척 잘 짜줍니다.


>

🗣️ 프롬프트(명령어)의 중요성

AI에게 원하는 것을 시킬 때는 구체적이고 명확하게 말해야 합니다.

❌ 나쁜 예시 "춤추는 외계인이 걸어가다 넘어지는 동영상 만들어 줘"
⭕ 좋은 예시 "메타몽 같이 생긴 귀여운 외계인이 끈적끈적하게 걸어가다가 철푸덕하고 넘어지는 동영상을 10초 정도로, 고화질로 만들어줘"

이렇게 원하는 바를 자세히 전달해야 빠르고 정확한 결과를 얻을 수 있습니다. 코딩도 마찬가지입니다!

실행과 오류 해결

🚀 웹사이트 실행

VSC 오른쪽 아래의 'Go Live' 버튼을 누르세요. 그러면 브라우저가 켜지면서 내가 만든 사이트가 보입니다. (아직은 내 컴퓨터에서만 보입니다!)

🔧 오류가 났다면? (F12의 마법)

아무리 프롬프트를 잘 짜도 AI는 완벽하지 않습니다. 오류 해결은 이렇게 하세요:

  1. 웹사이트에서 키보드 F12 키를 누릅니다.
  2. 나오는 창에서 '콘솔(Console)' 탭을 클릭합니다.
  3. 빨간색으로 뜬 오류 메시지가 있다면 그대로 복사합니다.
  4. AI에게 "이런 오류가 떴는데 수정해 줘"라고 하면 뚝딱 해결해 줍니다.

배포 및 추가 팁

내 컴퓨터에만 있는 사이트를 남들도 볼 수 있게 하려면 배포를 해야 합니다.

기본적으로 GitHub Pages를 추천하지만, 내용이 길어지니 유튜브나 AI에게 "깃허브로 웹사이트 배포하는 법"을 물어보시는 것을 추천합니다.

✨ 추천 도구 & 채널
  • Firebase: 데이터베이스, 로그인 등 고급 기능 (개발 후반부 추천)
  • Auth0: 간편한 로그인 기능 구현
  • 유튜브 채널: 코딩애플 등

마지막으로 드리고 싶은 말

개발은 솔직히 지루하고 힘듭니다. 오류를 고치는 건 정말 짜증 나는 일이죠.

하지만 우리는 목표물을 만들기 위해 여기까지 달려왔잖아요? 끈기를 가지고 하나씩 해결해 나갑시다. 오늘 안 되면 내일 하면 됩니다.

저도 처음엔 '친구 놀리기 사이트', '노래방 사이트' 같은 작은 것부터 시작했습니다. 그런 경험들이 쌓여 해정뉴스가 되었습니다. 포기하지 말고 끊임없이 도전해 보세요!
